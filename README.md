# Motion Graph – Wedding Photography Portfolio

A stunning, minimal wedding photography portfolio showcasing cinematic love stories.  
Built with **Next.js**, **Tailwind CSS**, and **Framer Motion** for smooth animations and full SEO.

![Motion Graph Screenshot](https://via.placeholder.com/800x400?text=Motion+Graph+Preview)

## 🎯 Features

- 🎥 **Video hero background** – immersive cinematic introduction
- 🖼️ **Dynamic gallery** – filterable masonry grid with lightbox
- 💬 **Floating WhatsApp button** – instant contact
- 📱 **Fully responsive** – flawless on mobile, no horizontal scroll
- 🔍 **SEO optimized** – meta tags, Open Graph, Schema.org structured data
- 🎞️ **Smooth animations** – page transitions, scroll reveals, hover effects
- 🌙 **Dark elegant theme** – cream and gold accents

## 🚀 Tech Stack

- [Next.js](https://nextjs.org/) (React framework)
- [Tailwind CSS](https://tailwindcss.com/) (utility-first CSS)
- [Framer Motion](https://www.framer.com/motion/) (animations)
- [React Icons](https://react-icons.github.io/react-icons/) (icons)
